== Description ==

The Cobol Press Post Types plugin, is a plugin to make Custom Post Type creation and development easier and faster!

With the Cobol Press Post Types plugin, you can generate as many Post Types as you want with the click of a button.

== Installation ==

1. Download Cobol Press Post Types
2. Install via ftp by unzipping and uploading or install directly through the plugins page.
3. Activate the plugin and Click on the new menu item “Cobol Types”
4. From this admin menu you can create and delete new Custom Post Types!
